📝 Project Story

How the Manifest Fashion E-commerce Dashboard was built from data to decision-making.

🧩 Problem Statement

In today’s fast-paced fashion e-commerce environment, businesses generate massive volumes of customer, sales, and return data across multiple platforms and payment methods. However, this data often remains underutilized due to lack of proper integration, analysis, and reporting. As a result, key questions remain unanswered:

  • 📦 What products and categories are driving the highest sales and returns?
  • 🕐 Which months or regions generate peak orders and revenue?
  • 💳 Which payment methods do customers prefer the most?
  • 🔁 What is the impact of discounts on return rates?
  • 📊 How can we visualize trends and KPIs in an accessible and interactive way?

This project aims to build a complete end-to-end Fashion E-Commerce Analytics Solution — from raw data to impactful dashboards — that helps answer such business questions and empowers data-driven decision-making.

📦 Dataset Overview

  • 🧑‍💼 customers.csv: Customer demographics like age, gender, region, and signup date
  • 🛒 orders.csv: Order-level data including product ID, quantity, discount, and payment method
  • 📦 products.csv & categories.csv: Product name, category, and pricing
  • ↩️ returns.csv: Return date and reason for each returned order

🛠️ Tools & Workflow

SQL

Created relational schema, loaded CSVs, and performed data aggregation and joins.

Python

Used Pandas, Seaborn, and Matplotlib for data cleaning and EDA.

Excel

Created PivotTables, slicers, and KPI cards for business dashboards.

Power BI

Designed modern, filterable dashboards to highlight customer & sales trends.

🎯 Final Outcome

The Manifest project showcases end-to-end data analytics proficiency — combining data engineering, exploration, visualization, and web development. It demonstrates the ability to solve business problems through structured analysis, storytelling, and dashboard building. The entire process is hosted and accessible via an interactive website.
